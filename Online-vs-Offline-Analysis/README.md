# 🎓 Online vs Offline Learning Analysis

## 📌 Project Overview
This project performs an **Exploratory Data Analysis (EDA)** on a dataset comparing online and offline learning environments. The goal is to identify patterns in student performance, study habits, and focus levels.

## 🛠️ Tools Used
- **Python**: Core programming language.
- **Pandas**: Data manipulation and cleaning.
- **Matplotlib/Seaborn**: Data visualization.
- **NumPy**: Numerical operations.

## 🔍 Key Insights
- Correlation between **Study Hours** and **Exam Scores**.
- Performance distribution across different **Learning Modes**.
- Impact of **Focus Level** on retention.

## 📂 Project Structure
- `online_vs_offline_learning_dataset.csv`: The raw dataset.
- `Analysis.ipynb`: The Jupyter notebook containing the full analysis.
